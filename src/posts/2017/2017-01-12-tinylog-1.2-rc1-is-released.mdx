---
title: tinylog 1.2-rc1 is released
---

The first release candidate of tinylog 1.2 is released and can be [downloaded](/v2/download/). Thereby tinylog 1.2 is feature complete. From now on, there will only bug fixes until the major release. As with tinylog 1.0 and 1.1, the first release candidate without any reported bugs will be released unmodified as final tinylog 1.2.

Changes:

* Configurations can be defined as URL to load them from network or web
* Support for choice format in log messages
* Support for custom number formatting in log messages
