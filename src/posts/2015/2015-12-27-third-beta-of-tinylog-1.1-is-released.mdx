---
title: Third beta of tinylog 1.1 is released
---

The third beta version of tinylog 1.1 provide an optional binding for Apache Commons Logging (JCL). Log entries from frameworks or libraries, which use Apache Commons Logging, can be output via tinylog if the new JCL binding is added to the classpath. The Spring Framework uses Apache Commons Logging for example. The beta version of tinylog and the new JCL binding are available for [download](/v2/download/).
